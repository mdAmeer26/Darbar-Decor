# Darbar Decor Frontend

Modern, responsive React application for Darbar Decor event decoration company.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

- **Gallery** - Filterable image gallery

## Customization

### Colors
Edit `tailwind.config.js` to change color palette.

### Content
Edit page files in `src/pages/` directory.

### Components
Reusable components in `src/components/` directory.

### API Endpoint
Update API base URL in components that use axios.
